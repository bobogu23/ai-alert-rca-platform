# AI 告警自动归因分析系统（Alert RCA Copilot）

## 项目定位

面向生产告警场景的 AI 自动归因分析平台。系统接收告警后，自动完成 CLS 日志检索与去重聚类、调用链梳理、伽利略 Metric/Trace 瓶颈分析，最终生成"一眼看透原因与证据"的归因报告，替代人工手动查日志、顺链路、翻监控的低效流程。

## 当前阶段

**设计阶段**。本仓库当前仅包含系统设计文档，代码尚未开始编写，待设计文档定稿后再进入开发阶段。

## 核心能力概览（第一版范围）

- 告警接入、指纹去重与抑制、级联告警合并。
- CLS 日志检索、指纹聚类、框架堆栈裁剪、调用链梳理。
- 伽利略 Metric/Trace 瓶颈定位（接口耗时高场景）。
- 多 Agent 根因推理与证据链校验（防幻觉）。
- 结构化、可追溯的归因报告自动生成，通过可扩展的 `NotificationGateway` 推送（当前仅启用企业微信渠道）。

> 历史相似案例检索（RAG）经评估后从第一版范围移除，架构已预留扩展点，后续迭代再评估引入，详见详细设计文档第 6.4 节。

## 技术栈概要

Java 21 + Spring Boot 3 + Spring AI 2.0（AI 框架首选） + Java Orchestrator 编排（第一版不引入 Dify） + 七彩石配置化路由 + LiteLLM 网关（大模型接入） + MCP（CLS/伽利略工具接入），详见详细设计文档第 1 章。
